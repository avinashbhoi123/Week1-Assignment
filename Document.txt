2. Explain the purpose of comments in HTML and provide an example of how to use comments in an HTML document?
The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date..

4. Write a short note on Tag and element with an example?
Each element is an individual component on a particular web page or document. They are in every part of the page, including the header, footer, body, and text. The format of a tag is a start tag followed by the content and an end tag, eventually.
5. What is the DOCTYPE Declaration in HTML?
The HTML document type declaration, also known as DOCTYPE , is the first line of code required in every HTML or XHTML document. The DOCTYPE declaration is an instruction to the web browser about what version of HTML the page is written in. This ensures that the web page is parsed the same way by different web browsers.
